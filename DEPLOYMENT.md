## Easiest Method: GitHub Pages (FREE)

### Step 1: Get Your Files
Download all the Crystal files from this conversation:
- index.html
- styles.css
- app.js
- manifest.json
- sw.js
- icon-192.svg
- icon-512.svg
- README.md

### Step 2: Create GitHub Account & Repository
1. Go to **github.com** and sign up (free)
2. Click the **"+"** icon ‚Üí **"New repository"**
3. Name it: `crystal-app`
4. Make it **Public**
5. Click **"Create repository"**

### Step 3: Upload Your Files
1. On your repository page, click **"uploading an existing file"**
2. Drag all 8 Crystal files onto the page
3. Scroll down and click **"Commit changes"**

### Step 4: Enable GitHub Pages
1. Click **Settings** tab in your repository
2. Scroll to **"Pages"** section (left sidebar)
3. Under "Source", select **"main"** branch
4. Click **Save**
5. Wait 1-2 minutes for deployment
6. Your app URL will be: `https://[your-username].github.io/crystal-app`

### Step 5: Install on iPad
1. Open **Safari** on your iPad
2. Go to your GitHub Pages URL
3. Tap the **Share button** (‚¨ÜÔ∏è)
4. Tap **"Add to Home Screen"**
5. Name it "Crystal" and tap **Add**

### Step 6: First Use
1. Tap the Crystal icon on your home screen
2. Allow **Location** access
3. Allow **Notifications** (for alerts)
4. Tap **"+ Drop Crystal"** to create your first memory!

## üÜò Quick Troubleshooting

**App won't install?**
- Must use Safari browser
- Try adding from bookmark first, then to home screen

**Location not working?**
- Settings ‚Üí Safari ‚Üí Location ‚Üí Allow
- Settings ‚Üí Privacy ‚Üí Location Services ‚Üí ON

**No notifications?**
- iOS limits background location for web apps
- Use "Check In" button when you arrive places
- Or just open the app - it auto-checks!
